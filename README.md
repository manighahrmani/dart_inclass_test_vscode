# dart_inclass_test_vscode
Portable VS Code + Dart SDK for in-class tests.

## Quick Install
```powershell
irm https://github.com/manighahrmani/dart_inclass_test_vscode/releases/latest/download/dart_inclass_test.zip -OutFile $env:TEMP\d.zip; Expand-Archive $env:TEMP\d.zip ~/Desktop -Force; rm $env:TEMP\d.zip; ~/Desktop/dart_inclass_test/DOUBLE_CLICK_ME_TO_START_TEST.bat
```

Or download the zip from [Releases](https://github.com/manighahrmani/dart_inclass_test_vscode/releases) manually.
