# VS Code For Dart In-Class Tests

A portable VS Code with Dart SDK bundled for in-class programming tests.

## Quick Setup (Windows Only)

1. Open the start menu, search for "PowerShell"
2. Open Powershell, copy the command below and enter it there:
```plaintext
irm https://raw.githubusercontent.com/manighahrmani/dart_inclass_test_vscode/main/install.ps1 | iex
```
3. Wait for the script to download, extract and launch VS Code. **This might take a minute.**
3. Click "Trust" if prompted about trusting the folder.
4. Then read the instructions in the comments of `main.dart` to get started. **Wait a few seconds for the start button to appear.**

## Manual Setup

1. Download `dart_inclass_test.zip` from [Releases](https://github.com/manighahrmani/dart_inclass_test_vscode/releases/latest)
2. Extract the zip
3. Double-click `DOUBLE_CLICK_ME_TO_START_TEST.bat`
